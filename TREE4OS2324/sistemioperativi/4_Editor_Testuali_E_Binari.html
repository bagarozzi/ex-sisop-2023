   vi (solo come approfondimento )

       vi__comandi_principali.pdf i principali comandi dell' editor vi
       Guida su vi di Fiorella Sartori una guida piu approfondita

       eliminare l'orrida colorazione delle parole chiave bash, col comando :syntax off
       attivare l'orripilante colorazione delle parole chiave bash, col comando :syntax on
       attivare l'auto indentazione usando tab di dimensioni 2 caratteri, col comando :set ai sw=2 ts=2
       attivare l'evidenziazione di parentesi tonde e graffe corrispondenti a quelle digitate, col comando :set sm

       settare le opzioni di vi per un utente scrivendole nel file ~/.vimrc
       esempio ~/.vimrc

        una chicca: usare vim come editor binario esadecimale, col comando :%!xxd
                                 ritornare alla visualizzazione normale, col comando :%!xxd -r

        Personalizzare la colorazione di parole chiave: esempio di file con sintassi delle parole chiave Insulti.vim
